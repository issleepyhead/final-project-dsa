# FINAL-PROJECT-DSA
Final project for Data Structure and Algorithms

## Usage
Navigate to the root folder then run this command
```bash
java -jar app.jar
```

## Compiling using gradle
Navigate to the root folder then run this command
```bash
gradle build
```
